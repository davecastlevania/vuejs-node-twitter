## Tweet Clone

Uses the following 

- AdonisJS for Models, Database, and Backend
- VueJS for the Front End
- NodeJS

To run

### Backend
- Turn On MySQL Instance
- Create A DB "TweetDB"
- Copy .env.example into .env
- Then Follow the Following Steps

```
cd tweet-api
npm install
addonis migrate:run
npm run start
```

### Frontend
```
cd tweet-app
npm install
npm run start
```

Front-End is accessible at http://localhost:8080
